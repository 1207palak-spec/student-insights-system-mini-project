# Student Management System (Mini Project)

This is a Python-based Student Management System developed as a mini project. It allows users to manage student records efficiently with support for both single and multiple student data entry.

## 🚀 Features
- Add a single student with marks
- Add multiple students at once (batch input)
- Store student data using file handling (persistent storage)
- Display all student records
- Calculate average marks of students
- Find class topper

## 🛠 Technologies Used
- Python
- File Handling
- Data Structures (Dictionary, List)

## 📂 How It Works
- Student data is stored in a text file (`students.txt`)
- Data is automatically loaded when the program starts
- Any new student added is saved permanently

## ▶️ How to Run
1. Run the Python file
2. Choose options from the menu
3. Add student data (single or multiple)
4. View results and analysis

## 💡 Learning Outcomes
- Understanding of functions and modular programming
- Implementation of menu-driven systems
- Working with file handling for data persistence
- Managing and analyzing data using Python
